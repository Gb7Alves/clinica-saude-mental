Sobre o Sistema

Este sistema foi desenvolvido para gerenciar os cadastros e agendamentos de uma clínica de saúde mental, incluindo:

  • Cadastro de Pacientes: Registre informações completas dos pacientes, incluindo histórico médico.
  • Cadastro de Profissionais: Mantenha dados dos médicos e estagiários.
  • Registro de Consultas: Documente cada consulta realizada com diagnóstico e prescrição.
  • Agendamentos: Organize os horários de atendimento de forma eficiente.
