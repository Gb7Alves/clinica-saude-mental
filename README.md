🏥 Sistema de Gestão de Clínica de Saúde Mental

Sistema desenvolvido em Python com Flask e SQLite, para gerenciar pacientes, profissionais e consultas de uma clínica de saúde mental.

🌟 Funcionalidades: 🧑‍⚕️ Cadastro de pacientes, 👩‍⚕️ Cadastro de profissionais, 📋 Registro de consultas, 📅 Agendamento de consultas, 🔍 Visualização de informações de pacientes, profissionais e consultas.

🛠 Tecnologias: Python 3.x, Flask, SQLite, HTML / CSS (templates).

🚀 Como Rodar o Sistema: Clone o projeto com git clone https://github.com/Gb7Alves/clinica-saude-mental.git e entre na pasta com cd clinica-saude-mental. Instale as dependências com pip install -r requirements.txt. Rode o sistema com python run.py. Abra no navegador em http://127.0.0.1:5000. Para parar o sistema, pressione CTRL + C no terminal.

📂 Estrutura do Projeto: app/ contendo __init__.py, models.py, routes.py, templates/; clinica.db; requirements.txt; run.py.
